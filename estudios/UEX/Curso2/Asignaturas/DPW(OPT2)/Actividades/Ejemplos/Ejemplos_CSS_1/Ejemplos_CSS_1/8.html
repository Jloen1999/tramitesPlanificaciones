Haz click para mostrar más

Este panel contiene un contenedor div, oculto por defecto (display: none).

Mediante la propiedad CSS y JS conseguimos mostrarlo (display: block).

El párrafo con clase flip tiene asociado un atributo onclick que permite la llamada a la función mostrar cuando el usuario haga click en el párrafo (como si fuera un botón). La función implementada permite mostrar u ocultar el contenedor según el estado de su propiedad display.
